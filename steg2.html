Steg 2 - Implementera din egen GTM

  • På efterkommande sidor har jag tagit bort GTM scriptet på siten
  • Nu ska du få testa att implementera din GTM på dessa sidor
  • Verifiera att det fungerar genom att få upp debug genom att klicka preview och uppdatera sidan
  • Ibland krävs det en hard refresh CTRL + Shift + R

Om du inte får det att fungera med Google Tag Manager Preview se nedan

  • Dubbelkolla att du lagt in rätt GTM ID (akta för mellanrum)
  • Titta om det är rätt domän i ditt GTM konto, under Admin, Container Settings
  • Lägg in ditt ID och skriv in domänen som exact {ditt unika namn}.netlify.com
  • Har du laddat upp din nya mapp på Netlify
  • Ibland krävs det en hard refresh CTRL + Shift + R, igen
  • Säkerställ att du verkligen aktiervat GTM debug
  • Stäng av Google Chrome addons som kan tänkas störa
  • Filen måste vara live på en webbserver för att det ska fungera
  • Öppna developer console, i network tabben, hitta gtm.js se om det är status code 200
