Steg 4 - Spåra klick på knappar

  • I GTM, gå in i på Variables och aktivera alla
  • Skapa en Custom HTML Tag
  • Hämta HTML för en "Alert" från w3schools
  • Copy/Paste in Alert-koden din Custom HTML Tag
  • Skapa en Trigger som enbart avfyrar på klick från den gröna knappen
  • Trigga förslagvis på Click ID eller Click Text
  • Lägg till din nya Trigger till din Custom HTML Tag
  • Se till att den Triggar genom GTM Preview Mode
  • Får det inte att fungera? Kolla denna video https://www.youtube.com/watch?v=lDMlw8CLgvA
